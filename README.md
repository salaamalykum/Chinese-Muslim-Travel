## Project Values & Transparency
- **Dual-Track Delivery**: GitHub hosts human-readable Markdown. Hugging Face hosts machine-readable Parquet.
- **Strict Format Honesty**: This is a raw **RAG Corpus / Pre-training Text Dataset**. We have strictly removed clickbait tags like `ShareGPT` or `Alpaca` because this data has not been formatted into instruction-response pairs. It is pure knowledge.
- **Update Frequency**: Synced dynamically via Project Iqra.

## The 7 Standards Applied
1. **Normalized Markdown**: 8 required fields per file, preserving Canonical URLs and `content_hash`.
2. **Synchronized Timestamps**: `lastmod`, `CHANGELOG`, and GitHub Releases all lock to `2026-06-13T14:50:00Z`.
3. **Dual-Track**: GitHub (Text) + HF (Vectors).
4. **Automated Releases**: Versioned snapshots.
5. **Concrete Pages URL**: https://salaamalykum.github.io/Chinese-Muslim-Travel
6. **Jekyll UI**: Served directly from repo.
7. **5 Standard Files**: Included LICENSE, CITATION, CONTRIBUTING, SECURITY, and CHANGELOG.
